# Monagotchi

Your very own virtual Mona pet on your Badger 2350! Take care of your digital companion by playing, feeding, and keeping them clean.

## Features

- **Feed Your Pet**: Keep Mona well-fed and happy
- **Play Games**: Interact and play with your virtual companion
- **Keep Clean**: Maintain hygiene by cleaning your pet regularly
- **Happiness Meter**: Monitor Mona's mood and well-being
- **Growth System**: Watch your pet grow over time
- **Persistent State**: Your pet's stats are saved between sessions

## Tips for a Happy Pet

- Check on your Monagotchi regularly throughout the day
- Stats decrease over time, so don't neglect your pet!
- Playing increases happiness the most
- Feeding when not hungry won't help
- A clean pet is a happy pet
- Your pet sleeps at night (no need to wake them!)

## Code Example

Basic pet care system:

```python
from badger2350 import Badger2350
import time

badge = Badger2350()

class Monagotchi:
    def __init__(self):
        self.hunger = 50
        self.happiness = 80
        self.cleanliness = 100
    
    def feed(self):
        if self.hunger > 0:
            self.hunger = max(0, self.hunger - 20)
            self.happiness += 5
    
    def play(self):
        self.happiness = min(100, self.happiness + 15)
        self.hunger += 5
        self.cleanliness -= 10
    
    def clean(self):
        self.cleanliness = 100
        self.happiness += 10
    
    def display(self):
        badge.set_pen(15)
        badge.clear()
        
        badge.set_pen(0)
        badge.text("Monagotchi", 40, 10, scale=2)
        badge.text(f"Hunger: {self.hunger}", 20, 50)
        badge.text(f"Happy: {self.happiness}", 20, 70)
        badge.text(f"Clean: {self.cleanliness}", 20, 90)
        
        badge.update()
```

## Code Example

Here's how the pet state is managed:

```python
from badger2350 import Badger2350
import json
import time

badge = Badger2350()

class Monagotchi:
    def __init__(self):
        self.hunger = 100
        self.happiness = 100
        self.cleanliness = 100
        self.last_update = time.time()
    
    def update_needs(self):
        # Needs decrease over time
        time_passed = time.time() - self.last_update
        hours = time_passed / 3600
        
        self.hunger = max(0, self.hunger - hours * 10)
        self.happiness = max(0, self.happiness - hours * 5)
        self.cleanliness = max(0, self.cleanliness - hours * 3)
        
        self.last_update = time.time()
    
    def feed(self):
        self.hunger = min(100, self.hunger + 30)
    
    def play(self):
        self.happiness = min(100, self.happiness + 25)
    
    def clean(self):
        self.cleanliness = min(100, self.cleanliness + 40)
```
