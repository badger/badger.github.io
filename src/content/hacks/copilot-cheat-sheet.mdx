# GitHub Copilot Cheat Sheet

Turn your Badger 2350 into a portable reference guide for GitHub Copilot! This app displays a comprehensive cheat sheet covering Copilot features, slash commands, chat participants, and best practices.

## What You'll Learn

- Text file reading and parsing
- Multi-page text rendering
- State management and persistence
- Font and layout customization
- Navigation interface design

## Hardware Requirements

- Badger 2350
- USB-C cable for programming

## How to Use

- **UP/DOWN Buttons**: Navigate between pages
- **A Button**: Change font size
- **B Button**: Change font style
- **A+C**: Exit to launcher

## Features

### Multi-Font Support
Choose from 4 different font styles:
```python
FONTS = ["sans", "serif", "bitmap8", "cursive"]
THICKNESSES = [1, 2, 2, 1]
```

### Smart Text Rendering
The app intelligently wraps text and handles pagination:
```python
def render_page():
    row = 0
    line = ""
    pos = ebook.tell()
    
    while True:
        words = ebook.readline().split(" ")
        next_word = words[0]
        
        # Check if adding this word would exceed line width
        if len(line + " " + next_word) * text_size > TEXT_WIDTH:
            # Start new line
            display.text(line, TEXT_PADDING, row * text_spacing, WIDTH, text_size)
            line = next_word
            row += 1
```

### State Persistence
Your reading position and preferences are automatically saved:
```python
state = {
    "last_offset": 0,
    "current_page": 0,
    "font_idx": 0,
    "text_size": 0.5,
    "offsets": []
}
badger_os.state_save("ebook", state)
```

